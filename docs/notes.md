# Runtime Notes

Local run command:

```powershell
python app.py
```

Default local address:

```text
http://127.0.0.1:8765
```

Useful settings:

- `MAX_UPLOAD_BYTES`: upload size limit in bytes.
- `FILE_RETENTION_HOURS`: source and export retention hours.
- `REDACT_EXPORTS`: set to `true` to mask exported reports.
- `RULE_PROFILE`: selected rule profile.
